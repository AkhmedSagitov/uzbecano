# Third-party extensions

We use some standard third-party extensions for our project.
They should be installed for any of our projects but are not part of the
kickstarter by default to keep the template repository small and easy to update.

- [Auth0](https://extensions.typo3.org/extension/auth0)
- [Content Defender](https://extensions.typo3.org/extension/content_defender)
- [Form to database](https://extensions.typo3.org/extension/form_to_database)
- [Frontend Editing](https://extensions.typo3.org/extension/xima_typo3_frontend_edit)
- [Mailcatcher](Mailcatcher/Index.md)
- [Sentry Client](https://extensions.typo3.org/extension/sentry_client)
- [T3 Monitoring Client](https://github.com/georgringer/t3monitoring_client)
- [TYPO3 Console](https://extensions.typo3.org/extension/typo3_console)
