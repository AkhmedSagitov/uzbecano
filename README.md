# TYPO3 Kickstarter project

Get going quickly with our standards for TYPO3 projects.
This repository is designed to be used as a template for new TYPO3 projects and a living documentation of our project standards.
Changes to our structural or technical standards must be discussed within the team and documented in this project.

The full documentation for the kickstarter project can be found [here](Documentation/Kickstarter/Index.md).
